# 🎬 Seatify

A modern online movie ticket booking web application built using **Django**.

Seatify allows users to browse movies, select seats, book tickets, download PDF tickets, receive email confirmations, and manage bookings through a clean and responsive interface.

---

# 🚀 Features

### 👤 User Module
- User Registration & Login
- Profile Management
- Booking History
- Cancel Booking
- Email Ticket Confirmation

### 🎬 Movie Module
- Movie Listing
- Search Movies
- Movie Details
- Coming Soon Movies
- Movie Reviews & Ratings

### 🎟 Booking Module
- Interactive Seat Selection
- Seat Availability
- Payment Summary
- PDF Ticket Download
- Booking Cancellation

### ⭐ Review System
- Add Reviews
- Movie Ratings

### 📊 Admin Module

- Revenue Summary

### 🎨 UI

- Dark Theme

- Interactive Dashboard

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Seatify.git
```

Create virtual environment

```bash
python -m venv env
```

Activate

Windows

```bash
env\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run migrations

```bash
python manage.py migrate
```

Start server

```bash
python manage.py runserver
```

Open

```
http://127.0.0.1:8000
```
